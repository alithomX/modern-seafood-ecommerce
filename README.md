# 🐟 Modern Seafood E-commerce Website

A modern and responsive seafood online store built with a focus on clean UI/UX, smooth user experience, and realistic e-commerce features.

This project simulates a real-world seafood marketplace where users can explore products, manage their shopping cart, and complete the ordering process.

## ✨ Features

### 🛒 Shopping Experience
- Product catalog
- Product categories
- Product details page
- Add to cart functionality
- Increase / decrease product quantity
- Remove products from cart
- Dynamic cart total calculation
- Dynamic Shopping Cart


### 🔍 Product Discovery
- Product filtering
- Category filtering
- Search functionality
- Product cards with detailed information

### 📦 Order System
- Checkout page
- Customer information form
- Shipping cost calculation
- Order summary
- Order confirmation flow

### 🎨 UI/UX
- Modern seafood-themed design
- Fully responsive layout
- Mobile-friendly interface
- Clean navigation
- Smooth interactions
- User-focused shopping experience
